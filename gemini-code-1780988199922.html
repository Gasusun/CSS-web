CSS (Cascading Style Sheets) là ngôn ngữ định dạng giúp biến những khung xương HTML thô sơ thành các trang web sống động, đẹp mắt và chuyên nghiệp. Hãy cùng điểm qua những khái niệm cốt lõi nhất.

1. Khái niệm Box Model

Trong CSS, mọi phần tử HTML đều được coi là một "chiếc hộp" (box). Mô hình chiếc hộp (Box Model) là nền tảng để tính toán kích thước vật lý của phần tử, bao gồm 4 thành phần từ trong ra ngoài:

  • Content: Vùng chứa nội dung hiển thị (văn bản, hình ảnh).
  • Padding: Khoảng đệm trống giữa nội dung và viền.
  • Border: Đường viền bao quanh lớp padding và content.
  • Margin: Khoảng cách trống ngoài viền, dùng để đẩy các phần tử khác ra xa.

2. CSS Display & Layouts

Thuộc tính display quyết định cách một phần tử hành xử và sắp xếp trên trang web. Có 3 giá trị cơ bản và 2 hệ thống Layout hiện đại:

Block

Luôn bắt đầu ở dòng mới và chiếm trọn 100% chiều rộng có thể. Ví dụ: <div>, <p>.

Inline

Chỉ chiếm vừa đủ không gian của nội dung, không xuống dòng, không đặt được width/height. Ví dụ: <span>, <a>.

Inline-Block

Nằm trên cùng một dòng như inline nhưng vẫn có thể tùy chỉnh kích thước width, height, padding, margin như block.

Các hệ thống Layout dàn trang hiện đại:

Flexbox (1 chiều)

Lý tưởng để sắp xếp các phần tử theo hàng ngang hoặc cột dọc, căn chỉnh khoảng cách linh hoạt giữa các phần tử.

CSS Grid (2 chiều)

Mạnh mẽ trong việc chia bố cục trang web phức tạp theo cả hàng (rows) và cột (columns) cùng lúc.

3. Thiết kế Responsive (Responsive Design)

Responsive Design là kỹ thuật giúp trang web tự động thay đổi kích thước, bố cục để **hiển thị tối ưu trên mọi thiết bị** (từ màn hình máy tính lớn, máy tính bảng đến điện thoại di động).

Công cụ cốt lõi của Responsive là Media Queries, cho phép áp dụng các đoạn mã CSS khác nhau dựa trên kích thước màn hình cụ thể.

// CSS thuần: Thay đổi màu nền khi màn hình nhỏ hơn 768px

@media (max-width: 768px) {

body {

background-color: lightblue;

}

}

4. Kỷ nguyên mới với CSS Frameworks

Để tăng tốc độ phát triển dự án, các lập trình viên thường sử dụng các Framework CSS đã được xây dựng sẵn. Hai "ông lớn" phổ biến nhất hiện nay đại diện cho hai triết lý thiết kế khác nhau:

B

Bootstrap

Triết lý Component-driven (Dựa trên thành phần). Cung cấp sẵn các UI component hoàn chỉnh (như Buttons, Cards, Modals, Navbar). Bạn chỉ cần gọi tên class.

<button class="btn btn-primary">Bootstrap Button</button>

Tailwind CSS

Triết lý Utility-first (Tiện ích trước tiên). Không cung cấp sẵn giao diện component nào, thay vào đó là hàng ngàn class nhỏ, nguyên tử (như bg-red-500, pt-4) để bạn tự do lắp ghép giao diện tùy ý mà không rời mắt khỏi file HTML.

<button class="bg-blue-500 hover:bg-blue-700 text-white px-4 py-2 rounded">Tailwind Button</button>
