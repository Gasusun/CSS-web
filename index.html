Làm Chủ Diện Mạo Website Với CSS

CSS (Cascading Style Sheets) là ngôn ngữ định kiểu quyết định diện mạo trang web. Hãy bấm vào các nút menu ở trên để xem chi tiết từng phần, các phần khác sẽ tự động tối đi và mini-game trắc nghiệm sẽ lộ diện!

1.

Giới thiệu Box Model

Trong CSS, mọi phần tử HTML đều được bao bọc bởi một khối hình hộp chữ nhật gọi là Box Model. Khối này được chia thành 4 lớp từ trong ra ngoài: Content (Nội dung), Padding (Khoảng đệm), Border (Đường viền), và Margin (Khoảng cách lề).

Content Nội dung thực tế: Chứa chữ, hình ảnh, video.

Padding Khoảng đệm: Khoảng cách giữa nội dung và đường viền.

Border Đường viền: Lớp viền bao quanh phần tử.

Margin Khoảng cách lề: Khoảng cách giữa phần tử đó và xung quanh.

Tổng width = Content + 2×Padding + 2×Border + 2×Margin

2.

Layouts (Bố cục)

Layout là cách sắp xếp và căn chỉnh các phần tử trên trang web. Các kỹ thuật layout phổ biến nhất hiện nay bao gồm:

Flexbox (Flexible Box Layout)

Tối ưu cho bố cục một chiều (hàng ngang hoặc dọc). Cực mạnh về co giãn linh hoạt.

CSS Grid Layout

Tối ưu cho bố cục hai chiều (cả hàng và cột cùng lúc), chia khối lớn trực quan.

3.

CSS Display

Thuộc tính display xác định cách phần tử hiển thị và chiếm không gian:

Block: Xuống dòng mới, chiếm 100% chiều rộng (div, p).
Inline: Cùng dòng, vừa vặn nội dung, không tùy chỉnh được kích thước (span, a).
Inline-Block: Cùng dòng nhưng đặt được width/height.
None: Ẩn hoàn toàn, không tốn diện tích trên màn hình.

4.

Responsive Design

Kỹ thuật giúp trang web tự động co giãn đẹp mắt trên mọi thiết bị (PC, Tablet, Mobile) chủ yếu thông qua Media Queries.

5.

CSS Frameworks: Bootstrap vs Tailwind

Bootstrap (Component-based)

Cung cấp sẵn UI hoàn chỉnh (.btn, .card, .navbar). Tạo mẫu trang web cực nhanh.

Tailwind CSS (Utility-first)

Cung cấp các lớp tiện ích nhỏ (bg-red-500, p-4). Tùy biến giao diện linh hoạt 100%.
